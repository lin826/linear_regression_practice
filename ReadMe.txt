
Run the main.py with python3 and add parameters as below:

--map_size    The size of map, for this homework is 1081.
--dim         The dimension of input vectors.
--data_size   The size of training data.
--batch_size  The size of batch run in gradient descent.
--iter        Total iterations to run in gradient descent.
--eta         The learning rate in per batch per iteration.
--k_folder    The number of validation folders for cross validation.
--basis_size  The number of basis functions.
--sigma       The varience of Gaussian(Normal) basis function.
--lamb        The lambda value in MAP approach.
--m0          M0 value in Bayesian approach.
--alpha       Alpha value in Bayesian approach.
--beta        Beta value in Bayesian approach.
--graph_scale Scale to resolution of predicted hight graph.
--x_train     Relative path to training data's input vector.
--t_train     Relative path to training data's output value.
